    1. RENOIR: Accelerating Blockchain Validation using State Caching , Nitin Awathare, Sourav Das, Vinay J. Ribeiro, Umesh Bellur, ICPE 2021
      Renoir speeds up the block validation in PoW Blockchains by state caching, which is evident by our measurements on the production Ethereum network that shows miners can speed up the block validation significantly using cached state.

    2. REBAL : Channel Balancing for Payment Channel Networks , Nitin Awathare, Suraj, Akash, Vinay J. Ribeiro, Umesh Bellur, MASCOTS 2021
      Rebal improves the transaction success ratio in the payment Channel Network by in-place re-balancing the channel that becomes unidirectional with time and reactively re-routes the transaction around the exhausting channel.

      Tuxedo scales the block validation by 100% by delaying execution of the transaction in performing it in parallel to the PoW.
